# Documentation

## Data :

### 1. Create a JSON

Create a file `yourFileName.json`

>[!IMPORTANT]
>Verify that this name don't exist already

Copy past this in the file :
```json
{
  "cardInfo":{
    "Title":"",
    "Text":"",
    "lang":"EN",
    "pictureURL":"None"
  },
  "type":[]
}
```

### 2. Edit basic info (needed)

**Change the language :**

If english : 
```json
"lang":"EN",
```
If french : 
```json
"lang":"FR",
```

**Add a title :**

Change the second half of this line :
```json
"Title":"Your title here",
```
**Q** : Is the name important ?

**A** : No, you can choose the name you want

**Add a text :**

Change the second half of this line :
```json
"Text":"Your text here",
```
**Q** : Is the name needed ?

**A** : No, you don't need one

**Add a picture :**

if you don't want a picture :
```json
"pictureURL":"None",
```

if you want one :
```json
"pictureURL":"https://fr.wikipedia.org/wiki/Uniform_Resource_Locator#/media/Fichier:Firefox_adress_bar_Uniform_Resource_Locator_on_frwiki.png",
```

>[!CAUTION]
>Only wikipedia source is accepted (for now)
