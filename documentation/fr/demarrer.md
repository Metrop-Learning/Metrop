## FAQ :

### Metrop fonctionne actuellement comme un site web statique afin d’être compatible avec GitHub Pages. Est-il possible que Metrop évolue vers un site web dynamique dans le futur ?

**Oui**, notre objectif est de passer à une architecture client–serveur, mais actuellement nous ne pouvons pas assumer le coût d’un serveur ni le temps nécessaire au développement de l’application côté serveur.

### Metrop disposera-t-il d’une application mobile ?

**Oui**, nous prévoyons de créer une application Android.
